# Aryan&apos;s Portfolio

## 🚀 Features
- **Modern Glassmorphism Design** - Beautiful backdrop blur effects and gradients
- **Dark/Light Theme Toggle** - Automatic persistence with localStorage
- **Fully Responsive** - Mobile-first design, works on all devices
- **Smooth Animations** - AOS scroll animations, hover effects, parallax
- **Interactive Elements** - Skill progress bars, project cards with tilt effects
- **Loading Screen** - Professional preloader with spinner
- **Contact Form** - Google Apps Script integration with validation
- **SEO Optimized** - Meta tags, semantic HTML, fast loading
- **Accessibility** - ARIA labels, keyboard navigation, reduced motion support

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Properties, Grid, Flexbox, Animations)
- **JavaScript**: Vanilla JS, AOS, Typed.js
- **Styling**: Glassmorphism, CSS Gradients, Backdrop Filters
- **CDNs**: AOS (Animations), Typed.js (Typing Effect)

## 📱 Preview
Open `index.html` in any modern browser to view the portfolio.

## 🎯 Sections
1. **Hero** - Animated typing effect, download resume, GitHub link
2. **Skills** - Interactive progress bars with hover effects
3. **About** - Professional bio with certifications & internships
4. **Projects** - 5 featured projects with Live Demo/GitHub links
5. **Testimonials** - Client feedback carousel
6. **Contact** - Working form with client-side validation

## 🚀 Quick Start
1. Clone/Download this repository
2. Open `index.html` in your browser
3. Toggle theme with the sun/moon icon
4. Test responsive design by resizing window

## 📝 Update Projects
Edit project descriptions and links in `index.html` > `#project` section.
